# RTL-to-GDSII-Traffic-Light-Controller-Using-Cadence-EDA-Tools
This project presents the complete ASIC design flow  of a Traffic Light Controller, from RTL coding in  Verilog to GDSII layout using Cadence EDA tools on  GPDK045 (45nm) technology. The design includes  RTL simulation, logic synthesis, DFT analysis,  place and route, static timing analysis, and  physical verification.
